### 느낀점

아직 익숙치 않은 React를 활용하여 비동기 통신을 통한 기능을 구현하려다 보니

버벅이는 부분이 많았던 것 같습니다.

특히나 jwt토큰을 처리하는데 있어 일단은 localStorage에 저장을 했지만, 찾아보니 localStorage에 저장하지 않고 다른 형식으로 하는 방법도 있었으며 그 방식이 보안적으로 더 우수했던 것 같아 추가적인 학습이 필요할 것 같습니다.
